### **nodeJs + mongodb 搭建 汽车名称增删改查**
<br>


###### 使用方式
1、$ cnpm i 安装依赖包 <br>
2、$ npm start 启动本地服务器 这里使用 node-dev（更改代码实时更新）或者 直接 $node app.js 就可以启动服务了  <br>
3、mongodb 启动
$ mongod --config /usr/local/etc/mongod.conf
$ mongo <br>
